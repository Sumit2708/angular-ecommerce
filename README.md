# Angular eCommerce Application

## Overview
A simple eCommerce web application built using Angular (Standalone Components)
as part of a technical assignment. The application focuses on clean UI,
modern Angular practices, and client-side state management.

## Features
- User login using FakeStore API
- Product listing with category filter
- Search products by title
- Product details page with rating and quantity selector
- Cart management using Angular Signals
- Quantity update and subtotal calculation
- Cart persistence using localStorage
- Route protection and invalid URL handling
- Clean UI aligned with provided Figma reference

## Tech Stack
- Angular 19 (Standalone Components)
- TypeScript
- SCSS
- Angular Signals
- FakeStore API

## Pages
- Login
- Products List
- Product Details
- Cart
